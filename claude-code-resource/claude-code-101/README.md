## 🧠 What is Claude Code?

**Claude Code** is Anthropic's AI coding assistant that runs directly in your terminal. Unlike browser chatbots, it works *inside* your actual project — reading files, writing code, running commands, and handling Git — all through natural language.

> Think of regular Claude as a consultant you email. Claude Code is that consultant sitting at your desk, doing the work while you watch.

**By the numbers (March 2026):**
- 📈 Responsible for **4% of all public GitHub commits** — ~135,000 commits per day
- 💰 Run-rate revenue surpassed **$2.5 billion**, doubling since January 2026
- 👥 Weekly active users **doubled since January 2026**
- 🏢 Enterprise customers now account for **more than half of total revenue**
- 🏗️ **90%** of Anthropic's own internal code is written with AI assistance

This tool is not the future. It's the present.

### Install & Start Claude Code
```bash
# Install Claude Code (use the native installer — NOT the old npm method)
# Visit: https://claude.ai/code and follow the native installer for your OS

# OR via npm (still works):
npm install -g @anthropic-ai/claude-code

# Navigate to any project folder
cd my-project

# Start Claude Code
claude
```

> ⚠️ **Note (March 2026):** Many older tutorials recommend `npm install -g @anthropic-ai/claude-code`. While it still works, the official docs now recommend using the **native installer** at [claude.ai/code](https://claude.ai/code) for better performance.

### Recommended VS Code Extensions
- **Claude Code** (official) — Inline diffs and context sharing
- **GitLens** — See exactly what Claude changed and when
- **Prettier** — Auto-formats code Claude writes

### System Requirements
- macOS 13.0+ (Intel or Apple Silicon)
- Windows 10 (version 1809+) or Windows 11
- Linux: Ubuntu 20.04+, Debian 12+
- Minimum 4 GB RAM + stable internet

## 🔑 Key Concepts to Learn

Master these and you'll move 10x faster than other beginners:

### 1. CLAUDE.md — Give Claude Permanent Memory
The most important thing to set up. A file that tells Claude about your project every session.
```bash
/init   # Auto-generates a CLAUDE.md in your project
```
Add to it: your tech stack, preferred coding style, how to run the app, key architecture decisions. Claude reads it automatically every session.

### 2. Plan Mode — Review Before Claude Acts
Always ask Claude to plan before making big changes:
```
/plan
Refactor the authentication module to use JWT tokens
```
Claude lists every file it will change, in order, and waits for your approval before touching anything.

### 3. Context Management — Keep Claude Sharp
Claude has a limited memory window per session:
- `/compact` — Summarizes the conversation to free up space (use around 70% full)
- `/clear` — Full reset, start fresh (use at 90%+)

### 4. Auto Mode — Skip Manual Approvals Safely
New in early 2026. Instead of clicking "approve" on every action, an AI classifier approves safe actions automatically and blocks dangerous ones.
```bash
claude --enable-auto-mode
# or toggle with Shift+Tab inside a session
```

### 5. Slash Commands Cheat Sheet
| Command | What It Does |
|---------|-------------|
| `/init` | Create CLAUDE.md project memory |
| `/plan` | Plan changes before executing |
| `/compact` | Compress context to free memory |
| `/clear` | Reset conversation completely |
| `/voice` | Toggle voice mode (push-to-talk) |
| `/loop 5m [task]` | Run a task on a recurring schedule |
| `/effort` | Set model effort level (low/medium/high) |
| `/model` | Switch between Claude models |
| `/help` | Show all available commands |

## 🆕 What's New in March 2026

Claude Code has shipped major updates almost every week this month. Here's what's new:

### 🎤 Voice Mode (`/voice`)
Type `/voice` to activate push-to-talk coding. Hold the spacebar to speak, release to send. Supports 20 languages. Rolling out to all users through March 2026.
> Covered by [TechCrunch](https://techcrunch.com/2026/03/04/anthropic-rolls-out-voice-mode-claude-code-xcxwbn/) · [Full breakdown](https://pasqualepillitteri.it/en/news/381/claude-code-march-2026-updates)

### 🔁 `/loop` — Scheduled Background Tasks
Run any prompt on a recurring interval — like a cron job inside Claude Code.
```bash
/loop 5m check the deploy        # checks deployment every 5 minutes
/loop 1h run the test suite       # runs tests every hour
```
> [Official Changelog](https://code.claude.com/docs/en/changelog)

### 🖥️ Computer Use + Remote Control
Claude can now control your Mac desktop — moving the mouse, clicking buttons, browsing the web. Pair it with Remote Control to send instructions from your phone while Claude executes on your computer.

### 📐 1 Million Token Context Window
Available on Max, Team, and Enterprise plans. Massive codebases can now be loaded into a single session.

### 🧠 Opus 4.6 as Default Model
Opus 4.6 is now the default model in Claude Code, with 64K output by default and up to 128K max.

### ⚡ New Commands
- `/effort` — Set reasoning depth (low / medium / high)
- `/simplify` — Simplify selected code
- `/batch` — Run operations across multiple files at once
- `/color` — Color-code your terminal session (useful for running multiple parallel sessions)
