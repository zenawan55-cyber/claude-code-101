# Contributing to Claude-Code-101

Thank you for helping keep this list useful and current!

---

## How to Add a Resource

1. Fork this repo
2. Edit `README.md`
3. Add your resource to the right section with the date
4. Open a Pull Request with title: `Add: [Resource Name]`

## What Makes a Good Resource

✅ Free or very affordable  
✅ Published or updated in 2025–2026  
✅ Beginner-accessible  
✅ Actually useful — not promotional content  

## What Doesn't Belong

❌ Paywalled content without a free tier  
❌ Outdated tutorials (pre-2024)  
❌ Promotional or spam content  
❌ Duplicate of something already listed  

---

## Found a Dead Link?

Open an issue labeled `broken-link` with the URL. Fixed fast.

---

Be kind. This is a learning community.
