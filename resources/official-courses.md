## 🏅 Anthropic Academy

**Link:** [anthropic.skilljar.com](https://anthropic.skilljar.com/)
**Cost:** Free

Anthropic's official learning platform. The most authoritative place to learn Claude Code — straight from the people who built it.

### Available Courses

| Course | What You'll Learn | Best For |
|--------|------------------|---------|
| **Claude Code in Action** | Architecture, multi-tool use, context management, custom commands, MCP integration, GitHub workflow automation | Developers with basic CLI + Git knowledge |
| **Claude Code: Skills** | Creating reusable Skills (.claude/skills/) to give Claude domain-specific knowledge | Intermediate users |
| **Claude Code: Sub-Agents** | Spinning up parallel AI agents that work simultaneously on different parts of a task | Intermediate-Advanced |
| **Model Context Protocol (MCP)** | Connecting Claude to external tools, databases, and APIs | Developers |
| **Claude API Development** | Building applications on top of Claude using the Anthropic API | Developers |

> **Tip:** Start with "Claude Code in Action" — it's the foundational course the others build on.

## 🟢 Free Course for Non-Coders

### CC for Everyone — Carl Vellotti
**Cost:** Free
**Format:** 11-lesson course taught *inside* Claude Code itself

The only Claude Code course designed specifically for people with no coding background. You learn to use Claude Code by actually using it — the lessons run inside the tool itself. No prior terminal experience required.

> **Perfect for:** Business owners, marketers, writers, and anyone who wants to use Claude Code without becoming a developer.

### CC for PMs
**Cost:** Free
**Format:** Written lessons

Same concept as CC for Everyone but tailored for Product Managers. Covers using Claude Code for PRD writing, user story generation, and technical spec creation.

## ✅ Recommended Learning Order

If you're starting from scratch:

1. **[Anthropic Academy — Claude Code in Action](https://anthropic.skilljar.com/)** — free, 1-2 hours
2. **[Official Quickstart Docs](https://docs.anthropic.com/en/docs/claude-code/quickstart)** — 15 min
3. **[Official Best Practices](https://docs.anthropic.com/en/docs/claude-code/best-practices)** — 30 min
4. Build something small using what you learned
5. Come back for the Skills, Sub-Agents, and MCP courses when ready to go deeper
