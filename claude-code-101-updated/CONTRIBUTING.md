# 🤝 Contributing to Claude-Code-101

Thank you for helping keep this list useful, accurate, and up-to-date! Every contribution — big or small — helps the next vibe coder who finds this repo.

---

## 📋 Ways to Contribute

### 1. Add a New Resource
Found a great tutorial, article, video, or tool not listed here?

1. Fork this repo
2. Find the right file inside `resources/` for your resource type:
   - New YouTube channel → `resources/youtube-channels.md`
   - New written article → `resources/written-guides.md`
   - New course → `resources/official-courses.md`
   - New tool or setup tip → `resources/tools-and-setup.md`
   - New Claude Code feature or update → `resources/march-2026-updates.md`
3. Add your resource following the existing format
4. Open a Pull Request with the title: `Add: [Resource Name] to [filename]`

### 2. Fix a Broken Link
Links go stale. If you find one:
- Open an Issue labeled `broken-link`
- Include: the broken URL, where it appears, and the correct URL if you know it

### 3. Improve an Existing Entry
Made a resource better with more context or a clearer description?
- Open a PR with title: `Update: [Resource Name]`
- Briefly explain what you improved and why

### 4. Suggest a New Category
Think we're missing an entire topic area?
- Open an Issue labeled `enhancement`
- Describe the category and list 2-3 resources that would go in it

### 5. Don't Know How to PR?
No problem — just open an Issue and describe the resource you want added. We'll add it and credit you in the CHANGELOG.

## ✅ What Makes a Good Resource

| ✅ Include | ❌ Skip |
|-----------|--------|
| Free or very affordable | Expensive paywalled content |
| Published or updated 2025–2026 | Outdated tutorials (pre-2024) |
| Genuinely beginner-accessible | Overly niche or advanced-only |
| Adds something new | Duplicates an existing entry |
| Real, working link | Broken or redirecting links |

---

## 📝 Format to Follow

When adding a written guide entry, use this format:

```markdown
### [Title of Resource — Source Name](https://link.com)
**Published:** Month Year | **Read time:** ~X min

One or two sentences describing what the reader will learn and who it's best for.

> 💡 Optional tip: when to read this, what makes it special.
```

When adding a YouTube channel, use this format:

```markdown
### [Channel Name](https://youtube.com/@handle)
**Skill Level:** Beginner / Intermediate / Advanced | **Update Frequency:** Weekly / Monthly

One paragraph describing the channel, content style, and who it's best for.

**Must-watch:** List 1-2 specific videos or series
```

## 🙏 Code of Conduct

Be kind. This is a learning community. Everyone here is figuring things out — from the person who just discovered Claude Code today to the developer who's been using it since launch. Respectful, constructive contributions only.
