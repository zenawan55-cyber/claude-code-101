## 🧠 What is Claude Code?

**Claude Code** is Anthropic's AI coding assistant that runs directly in your terminal. Unlike browser chatbots, it works *inside* your actual project — reading files, writing code, running commands, and handling Git — all through natural language.

> Think of regular Claude as a consultant you email. Claude Code is that consultant sitting at your desk, doing the work while you watch.

By early 2026, Claude Code was responsible for roughly **4% of all public GitHub commits** — around 135,000 commits per day. And 90% of Anthropic's own internal code is now written with AI assistance. This tool is not the future. It's the present.

### Install & Start Claude Code
```bash
# 1. Install Claude Code globally
npm install -g @anthropic-ai/claude-code

# 2. Navigate to any project folder
cd my-project

# 3. Start Claude Code
claude
```

- **GitLens** — See exactly what Claude changed and when

## 🔑 Key Concepts to Learn

Master these and you'll move 10x faster than other beginners:

### 1. CLAUDE.md — Give Claude Permanent Memory
The most important thing to set up. A file that tells Claude about your project every session.
```bash
/init   # Auto-generates a CLAUDE.md in your project
```
Add to it: your tech stack, preferred coding style, how to run the app, key architecture decisions.

### 2. Plan Mode — Review Before Claude Acts
Always ask Claude to plan before making big changes:
```
/plan
Refactor the authentication module to use JWT tokens
```
Claude lists every file it will change, in order, and waits for your go-ahead.

### 3. Context Management — Keep Claude Sharp
Claude has a limited memory window per session:
- `/compact` — Summarizes the conversation to free up space (use around 70% full)
- `/clear` — Full reset, start fresh (use at 90%+)

### 4. Slash Commands Cheat Sheet
| Command | What It Does |
|---------|-------------|
| `/init` | Create CLAUDE.md project memory |
| `/plan` | Plan changes before executing |
| `/compact` | Compress context to free memory |
| `/clear` | Reset conversation completely |
| `/model` | Switch between Claude models |
| `/help` | Show all available commands |

## 🗺️ Suggested Learning Path

### 🟢 Complete non-techie — never coded before
1. Watch [Nick Saraev's intro videos](https://youtube.com/@nicksaraev) — 30 min
2. Take [Anthropic's free official course](https://anthropic.skilljar.com/) — 1-2 hrs
3. Read [How to Use Claude Code](https://www.builder.io/blog/how-to-use-claude-code)
4. Build one small thing that solves a real problem you have

### 🔵 Used Claude.ai but never Claude Code
1. Read the [Official Quickstart](https://docs.anthropic.com/en/docs/claude-code/quickstart)
2. Watch Nick Saraev's 4-hour masterclass
3. Set up CLAUDE.md in an existing project
4. Let Claude refactor something you already built

### 🟣 Developer wanting to go deep
1. Complete all [Anthropic Academy](https://anthropic.skilljar.com/) courses
2. Explore MCP server integrations
3. Build a multi-agent workflow using sub-agents
4. Contribute to this repo with resources you discover!
