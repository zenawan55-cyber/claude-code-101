# Contributing to Awesome Claude Code

Thank you for helping keep this list useful and up-to-date!

---

## How to Add a Resource

1. Fork this repo
2. Edit `README.md`
3. Add your resource to the right section
4. Open a Pull Request with title: `Add: [Resource Name]`

## What Makes a Good Resource

✅ Free or very affordable  
✅ Published or updated in 2025–2026  
✅ Beginner-accessible  
✅ Actually useful — not just promotional content  

## What Doesn't Belong

❌ Paywalled content without a free tier  
❌ Outdated tutorials (pre-2024)  
❌ Promotional or spam content  
❌ Duplicate of something already listed  

---

## Found a Dead Link?

Open an issue labeled `broken-link` with the URL. We'll fix it fast.

---

Be kind. This is a learning community. Everyone here is figuring things out.
